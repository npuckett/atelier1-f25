Course Page Examples

Three real Canvas pages generated with this tool, alongside the plain-text prompts that produced them.

How to Read These Examples

Each example below shows a Canvas page type used in a real studio course (Atelier 1, Fall 2025). On the left is the plain-text prompt -- the markdown the instructor wrote. On the right is the Canvas-compatible HTML the agent produced.

These three page types cover most of what a Canvas course needs:

  • Course Homepage -- Course info, instructor, and full project schedule. Created once at the start of term.
  • Project Page -- One page per project or assignment. Contains the brief, constraints, steps, and linked assignments.
  • Class Page -- One page per class session. Contains the schedule for that day, resources, and next steps.

To use these prompts: Copy the markdown text, open a conversation with your AI agent (with SKILL.md active), paste the markdown, and ask it to generate a Canvas page. You can substitute your own course content into the same structure.

Using These Examples

These examples are starting points. To build your own version:

  1. Download the prompt for the page type you need (links above each example).
  2. Replace the content with your own course information -- keep the structure and layout notes at the bottom of each prompt.
  3. Open a conversation with your AI agent and make sure SKILL.md is active (either in .github/ for a local agent, or uploaded for a web agent).
  4. Paste your prompt and ask: "Transform this into a Canvas page."
  5. Copy the HTML output and paste it into Canvas via the HTML editor (</> button in the RCE toolbar).

Adapt the layout notes. The layout and color instructions at the bottom of each prompt are what give the agent its design direction. You can change these to match your own course's visual identity -- different colors, different column proportions, or different structural choices.

See the Methods section on the main page for guidance on building a reusable course template that carries your preferences automatically into every page.
