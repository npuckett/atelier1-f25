This final presentation is an opportunity to get feedback on your Thesis project from eachother and guests. Unlike the Playtests, this is presentation format and it should cover three basic areas:

  • Current State of the work
  • Development overview
  • Plan for GradEx (if participating)
Timing

Each project will have 15 minutes. Covering both presentation and questions/feedback.

Format

Create a presentation that covers the three main topics. Present the project in a format that assumes the audience has never seen the project before. We will be using all 4 screens, but looking at projects one at a time.
