Final Stretch Plan

Overview

At this point it is fairly simple. You have 2 weeks left in this term. Detail a plan from now to the final presentation.

Activity

Create a document that details your plan from now until the final presentation. This should include:

  • Note each day between now and the final presentation
  • Key milestones and features to be completed
  • If it is a group project, note this timeline for each person
Deliverable

A day-by-day plan from now to the final presentation, outlining key milestones and features. Group projects should include a timeline for each member.

Due

In-class submission
