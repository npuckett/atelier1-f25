Late Work Policy

Project Alt Controller
Overview

A critical part of any interactive project is to document it well. These works are always temporary, so it is important that you clearly capture the project for your portfolio. Documentation allows you to show the project in its best light, and also allows you to reflect on the process of making it. This isn't about proving that you made something, it is an opportunity for you to capture and archive the work.

Format

All documentation is to be created as a post on the course WordPress site.

WordPress Site - Log in using your OCADU credentials.

For guidance on how to create a post on WordPress, see the following tutorial: How to Make a Post on WordPress

Be sure to submit the URL of your post in Canvas.

Required Components

Your documentation post must include the following components:

Project Title

Title of your controller project.

Your Name

As you want it to appear on your portfolio.

Project Description (100 - 200 words)

  • Overall concept of the project
  • Simple description of how the controller works
  • How your custom switches are used to create a unique interaction
  • How your controller changes the standard experience of playing pong
  • NOTE: Don't give me unedited ChatGPT text. It always sounds ridiculous.

Hero Image

A high quality image of your controller being used. You should take care with setting up and photographing this interaction, don't just use a random photo from class. This image should tell the story of your project.

Video (10 - 20 seconds)

A video showing the controller in use. Embedded in the post using a service like YouTube or Vimeo.

  • Take care how/where you make the video. Shaky phone video in a random location is not acceptable. Consider lighting, background, framing, etc.
  • Don't add random music.

Process Photos (3 - 5 images)

Images showing how the project was made. These can include sketches, prototyping, testing, and final assembly.

Process Description (100 words)

A written description of how the project was made.
