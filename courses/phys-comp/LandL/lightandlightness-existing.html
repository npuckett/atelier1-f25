Description

Components Used: Arduino, Servo Motor, Light Sensor

This project focuses on creating kinetic sculptures that manipulate light and shadow. It will also focus on a principle of lightness through the choice of materials. The goal is create unconventional kinetic movements that evolve over time.

Create a kinetic sculpture that uses your servo motor to control a mechanism that is primarily made up of very lightweight materials: paper, vellum, fabric, etc. The goal to investigate how the mechanical, rotational movement of the servo can be transformed through code and mechanisms. In addition to the material itself, the sculpture should manipulate light and shadow in specific ways. The system will read input from a Light Sensor to affect and change its movement over time.

 

Tools

  • Input data: Light Sensor
  • Output
    • Minimum : one or more servo motors
      • Consider the strength of the motors
    • Optional : other actuators
      • LEDs, etc
  • Output (debugging): Arduino Uno R4 Wifi’s LED matrix
