Test 2.1 -- Definition list (dl/dt/dd)

Expected: Bold terms with indented definitions below each.

Rich Content Editor -- the WYSIWYG editor in Canvas for creating page content.
Module
A container in Canvas that groups pages, assignments, and files into units.
Front Page
The landing page students see when they open a Canvas course.

Test 2.2 -- Styled definition list (border accent)

Expected: Each term/definition pair has a left blue border accent.

Inline Style
CSS applied directly via the style attribute on an HTML element.
Semantic HTML
HTML that uses meaningful elements (details, figure, dl) rather than generic divs.

Test 2.3 -- Ordered list with start="5"

Expected: Numbering begins at 5 (5, 6, 7).

  1. This should be item 5
  2. This should be item 6
  3. This should be item 7

Test 2.4 -- Reversed ordered list

Expected: Numbering counts down (3, 2, 1).

  1. This should be item 3
  2. This should be item 2
  3. This should be item 1

Test 2.5 -- Ordered list type="A" (uppercase letters)

Expected: Items labeled A, B, C instead of 1, 2, 3.

  1. This should be item A
  2. This should be item B
  3. This should be item C

Test 2.6 -- Ordered list type="i" (lowercase roman numerals)

Expected: Items labeled i, ii, iii.

  1. This should be item i
  2. This should be item ii
  3. This should be item iii

Test 2.7 -- Nested mixed lists (ol > ul > ol)

Expected: Three levels of nesting with different marker types at each level.

  1. First numbered item
    • Bullet sub-item
      1. Lettered sub-sub-item a
      2. Lettered sub-sub-item b
    • Another bullet sub-item
  2. Second numbered item

Test 2.8 -- Custom list markers via list-style-type

Expected: Square markers, circle markers, and none (no markers).

Square:

  • Square marker item
  • Square marker item

Circle:

  • Circle marker item
  • Circle marker item

None:

  • No marker item
  • No marker item

Test 2.9 -- Emoji-prefixed checklist style

Expected: No bullet markers. Each item starts with a checkbox-like or status symbol.

  • ☑ Completed task (ballot box with check)
  • ☐ Incomplete task (empty ballot box)
  • ⚠ Warning item (warning sign)
  • ➤ Action item (arrow)
