Single Collapsible Element Test

Workshop 1 - Real Things, Fake Project

This is a test of a single collapsible element. It should be open by default when the page loads.

If this content is visible immediately when you load the page, then the 'open' attribute is working correctly. If you have to click the summary to see this content, then Canvas is removing or ignoring the 'open' attribute.

This test page contains a single details/summary element with the 'open' attribute. The content should be visible by default without clicking anything.
